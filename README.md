# みるぷらす — 家計簿KPIアプリ

月の目標を **週・日のKPIに自動で分解**して、「今このペースで大丈夫か？」が一目で分かるスマホ向け家計簿。外部依存ゼロの単一HTML + `localStorage` で動く、インストール不要のPWAです。

**▶ 使う:** https://mirupurasu-dev.github.io/mirupurasu/
（スマホのブラウザで開いて「ホーム画面に追加」すると、アプリのように使えます）

---

## できること

- **目標 → KPIブレイクダウン**：月予算を入れると **月間 / 週間 / 日間**（カテゴリ合算）に自動分解
  - 予算の決め方は2通り：`貯蓄から逆算`（手取り収入 − 目標貯蓄）／ `予算を直接指定`
- **ペース判定**：ホームを見た瞬間に「予算内ペース ✓ / オーバーペース ⚠」が分かる
  - 経過日数に対する基準線、**月末の着地予測**、「残り◯日・1日あと¥◯使える」を表示
- **貯蓄トラッキング**：今月の収入 − 支出、目標貯蓄への到達度、着地見込みと声かけ
- **かんたん入力**：自前テンキーで金額 → カテゴリ → 記録が1画面・親指1本で完結。保存後もその場に残り連続入力OK
- **カテゴリ内訳**：ドーナツ＋バーで「何に何円使ったか」を可視化
- **カテゴリ**：食べ物 / インフラ系 / 娯楽 / 飲み / 習い事 / 家賃 / 甲斐バンド
- ライト / ダーク自動対応、セーフエリア対応、データは端末内のみに保存

## 技術

- 単一 `index.html`（インラインCSS/JS/SVG、フレームワーク・ビルド・外部通信なし）
- 保存：ブラウザの `localStorage`（サーバー送信・アカウント登録なし）
- PWA：`manifest.webmanifest` ＋ アイコンでホーム画面に追加可能

## ローカルで動かす

```bash
# 任意の静的サーバーで開くだけ
python3 -m http.server 4720
# → http://localhost:4720
```

## プライバシー

入力した金額・カテゴリなどのデータは、すべて **お使いの端末のブラウザ内（localStorage）にのみ** 保存されます。外部サーバーへの送信は一切ありません。
